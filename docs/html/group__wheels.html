enum TurnWheelsTaskType

The type of tasks the wheels are currently executing.

Enumerator
NONE 

Wheel is stopped.

ANGLE 

Wheel is turning by a certain angle.

SPEED 

Wheel is turning by a certain speed.

SYNCHRONIZED 

Wheels are turning with a certain speed relative to each other.

TIMED_ANGLE 

Wheel is turning by a certain angle in a certain time.

SYNCHRONIZED_ANGLE 

Wheels are turning with a certain speed relative to each other and a certain angle.

TURN 

Robot is turning in place.
